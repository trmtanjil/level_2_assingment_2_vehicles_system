Vehicle Booking System API একটি ব্যাকএন্ড ভিত্তিক সিস্টেম, যেখানে ব্যবহারকারীরা নির্দিষ্ট তারিখ অনুযায়ী যানবাহন বুক করতে পারে।
এই সিস্টেমটি স্বয়ংক্রিয়ভাবে:

গাড়ির availability যাচাই করে

একই সময়ে একাধিক booking হতে দেয় না

মোট ভাড়া হিসাব করে

booking এবং vehicle status ম্যানেজ করে

ফিচারসমূহ

 JWT Authentication (Customer ও Admin role)

 Vehicle management system

 তারিখ অনুযায়ী booking

 overlapping booking প্রতিরোধ

 স্বয়ংক্রিয় total price calculation

 Vehicle availability tracking

 Customer নিজের booking দেখতে পারবে

 Admin সব booking দেখতে পারবে
